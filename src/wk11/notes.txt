Override: change implementation (one class method is overridden in another class)

Components:

Concrete class: a class that CAN be instantiated

Abstract class
    a dedicated super class
    can NOT be instantiated
    used to group liked components
    can have a method that is declared but has no default implementation
    internal: can ignore this internal concept
